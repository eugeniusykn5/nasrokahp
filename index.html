AHP Multi Expert

Sistem Pendukung Keputusan dengan metode AHP yang siap diposting ke GitHub

Aplikasi ini menghitung bobot prioritas kriteria, bobot alternatif per kriteria, rasio konsistensi, dan ranking akhir dari beberapa expert menggunakan agregasi rata-rata geometrik.

Konfigurasi Awal

Static HTML + JS

Skala AHP yang dipakai: 1 sampai 9. Sel diagonal selalu 1, dan sisi bawah matriks dihitung otomatis sebagai nilai kebalikan.
